Simple webApiREST para consumo de servicios, construido con [NestJS](https://github.com/nestjs/nest) framework + TypeScript + NodeJS

### Instalación

```bash
$ npm install
```

### Ejecutar API

```bash
# Desarrollo
$ npm run start

# Certificación
$ npm run start:dev

# Producción
$ npm run start:prod
```
